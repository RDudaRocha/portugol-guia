Vetores

Vetores são estruturas que permitem armazenar vários valores do mesmo tipo em uma única variável.

O que é um Vetor?

Um vetor é como uma lista de valores armazenados em sequência. Cada valor possui uma posição chamada índice.

Exemplo

Imagine um vetor chamado numeros com 5 posições.

numeros[0] numeros[1] numeros[2] numeros[3] numeros[4]

Declarando um Vetor

Para criar um vetor no Portugol utilizamos colchetes [ ].

Exemplo

inteiro numeros[5]

Isso significa que o vetor pode armazenar 5 números inteiros.

Atribuindo Valores

Podemos guardar valores em cada posição do vetor utilizando o índice.

Exemplo

inteiro numeros[5]

numeros[0] = 10
numeros[1] = 20
numeros[2] = 30
numeros[3] = 40
numeros[4] = 50

Usando Vetores com Repetição

Normalmente utilizamos loops para preencher ou percorrer um vetor.

Exemplo

programa
{
    funcao inicio()
    {

        inteiro numeros[5]
        inteiro i

        para (i = 0; i < 5; i++)
        {
            escreva("Digite um número: ")
            leia(numeros[i])
        }

        para (i = 0; i < 5; i++)
        {
            escreva(numeros[i])
        }

    }
}

Índices do Vetor

Primeira posição

Todo vetor começa no índice 0.

Última posição

A última posição será sempre tamanho - 1.

Exemplo

Um vetor de tamanho 5 vai de 0 até 4.
