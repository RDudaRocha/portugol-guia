Funções

Funções são blocos de código que realizam uma tarefa específica. Elas ajudam a organizar melhor o programa e evitar repetição de código.

O que é uma Função?

Uma função é um conjunto de instruções que pode ser chamado sempre que necessário.

Exemplo do dia a dia

Imagine uma função chamada prepararCafe().

Sempre que você chamar essa função, ela executará todos os passos necessários para preparar o café.

Estrutura de uma Função

Para criar uma função no Portugol utilizamos a palavra funcao.

Exemplo

funcao saudacao()
{
    escreva("Olá, seja bem-vindo!")
}

Chamando uma Função

Depois de criar uma função, podemos chamá-la dentro da função inicio().

Exemplo

programa
{

    funcao saudacao()
    {
        escreva("Olá!")
    }

    funcao inicio()
    {

        saudacao()

    }

}

Funções com Parâmetros

As funções podem receber valores chamados parâmetros.

Exemplo

funcao mostrarNome(cadeia nome)
{
    escreva("Olá ", nome)
}

Funções com Retorno

Algumas funções podem retornar um valor utilizando a palavra retorne.

Exemplo

funcao inteiro soma(inteiro a, inteiro b)
{
    retorne a + b
}

Exemplo Completo

programa
{

    funcao inteiro soma(inteiro a, inteiro b)
    {
        retorne a + b
    }

    funcao inicio()
    {

        inteiro resultado

        resultado = soma(5, 3)

        escreva("Resultado: ", resultado)

    }

}
