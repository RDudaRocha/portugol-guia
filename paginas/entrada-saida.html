Entrada e Saída de Dados

Em um programa precisamos interagir com o usuário. Para isso utilizamos comandos de entrada e saída de dados.

Saída de Dados

A saída de dados serve para mostrar informações na tela. No Portugol utilizamos o comando escreva().

Exemplo

programa
{
    funcao inicio()
    {

        escreva("Olá mundo")

    }
}

Mostrando Variáveis

Também podemos mostrar valores armazenados em variáveis.

Exemplo

programa
{
    funcao inicio()
    {

        cadeia nome
        nome = "Maria"

        escreva("Nome: ", nome)

    }
}

Entrada de Dados

A entrada de dados permite que o usuário digite informações. No Portugol utilizamos o comando leia().

Exemplo

inteiro idade

leia(idade)

Exemplo Completo

programa
{
    funcao inicio()
    {

        cadeia nome
        inteiro idade

        escreva("Digite seu nome: ")
        leia(nome)

        escreva("Digite sua idade: ")
        leia(idade)

        escreva("Seu nome é: ", nome)
        escreva("Sua idade é: ", idade)

    }
}

Boas Práticas

Peça informações claramente

Sempre mostre uma mensagem antes do leia() para o usuário saber o que deve digitar.

Organize as saídas

Utilize escreva() para mostrar mensagens claras e organizadas.

Use variáveis corretas

Certifique-se de usar o tipo correto de variável para cada informação.
