Introdução à Programação

Antes de aprender Portugol, precisamos entender o que é lógica de programação e algoritmos.

O que é um Algoritmo?

Um algoritmo é uma sequência de passos para resolver um problema.

Exemplo do dia a dia

1 - Acordar
2 - Escovar os dentes
3 - Tomar café
4 - Ir para a escola

O que é Lógica de Programação?

Lógica de programação é a forma de organizar pensamentos para resolver problemas usando computadores. Ela é a base para aprender qualquer linguagem de programação.

O que é Portugol?

Portugol é uma linguagem de pseudocódigo criada para ensinar programação de forma simples. Ela utiliza comandos parecidos com o português, facilitando o aprendizado.

Exemplo de código


programa
{
    funcao inicio()
    {
        escreva("Olá mundo")
    }
}
