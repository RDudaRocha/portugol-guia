Estruturas de Repetição

As estruturas de repetição permitem executar um bloco de código várias vezes. Elas são utilizadas quando precisamos repetir uma ação até que uma condição seja atendida.

O que é um Loop?

Um loop é uma estrutura que repete um conjunto de instruções enquanto uma condição for verdadeira.

Exemplo do dia a dia

Imagine que você precisa contar de 1 até 10.

Ao invés de escrever 10 vezes o mesmo comando, podemos usar um loop para repetir automaticamente.

Estrutura ENQUANTO

A estrutura enquanto executa um bloco de código enquanto a condição for verdadeira.

Exemplo

inteiro contador

contador = 1

enquanto (contador <= 5)
{
    escreva(contador)
    contador++
}

Estrutura PARA

A estrutura para é usada quando sabemos exatamente quantas vezes o código deve repetir.

Exemplo

para (inteiro i = 1; i <= 5; i++)
{
    escreva(i)
}

Diferença entre ENQUANTO e PARA

ENQUANTO

Usado quando não sabemos exatamente quantas vezes o loop irá repetir.

PARA

Usado quando sabemos exatamente quantas repetições serão necessárias.

Exemplo Completo

programa
{
    funcao inicio()
    {

        inteiro numero

        para (numero = 1; numero <= 10; numero++)
        {
            escreva(numero)
        }

    }
}
