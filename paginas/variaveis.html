Variáveis

Variáveis são utilizadas para armazenar dados dentro de um programa. Elas permitem guardar informações que podem ser usadas durante a execução do algoritmo.

O que é uma variável?

Uma variável funciona como uma "caixa" onde podemos guardar valores. Esses valores podem ser números, textos ou informações lógicas.

Exemplo do dia a dia

Imagine uma caixa com o nome idade.

Dentro dessa caixa podemos guardar um valor:

idade = 18

Tipos de Variáveis no Portugol

inteiro

Armazena números inteiros.

Exemplo: 1, 10, 25, 100

real

Armazena números com casas decimais.

Exemplo: 1.5, 10.75, 3.14

cadeia

Armazena textos.

Exemplo: nomes, frases ou palavras.

logico

Armazena valores verdadeiro ou falso.

Muito usado em decisões.

Declarando Variáveis

Antes de usar uma variável, precisamos declará-la informando seu tipo.

Exemplo

inteiro idade
real altura
cadeia nome
logico aprovado

Atribuindo Valores

Depois de declarar uma variável, podemos armazenar um valor nela utilizando o operador de atribuição.

Exemplo

inteiro idade

idade = 18

Exemplo Completo

programa
{
    funcao inicio()
    {

        cadeia nome
        inteiro idade

        nome = "João"
        idade = 18

        escreva("Nome: ", nome)
        escreva("Idade: ", idade)

    }
}

Pratique

2️⃣ Crie uma variável chamada idade
3️⃣ Atribua valores para elas
