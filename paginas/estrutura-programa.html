Estrutura de um Programa em Portugol

Todo programa em Portugol possui uma estrutura básica que organiza o código e define onde a execução começa.

Estrutura Básica

Todo programa começa com a palavra programa e possui uma função principal chamada inicio(), onde o código será executado.

Exemplo

programa
{
    funcao inicio()
    {
        escreva("Olá mundo")
    }
}

Explicação da Estrutura

programa

Define o início do programa. Todo código em Portugol deve estar dentro dessa estrutura.

funcao inicio()

É a função principal do programa. Quando executamos o código, o Portugol começa a rodar a partir daqui.

{ }

As chaves delimitam blocos de código. Tudo que estiver dentro delas faz parte daquela estrutura.

Fluxo de Execução

Quando o programa é executado, o Portugol segue os passos abaixo:

1️⃣ O programa é carregado

2️⃣ A função inicio() é chamada

3️⃣ As instruções dentro da função são executadas na ordem

Exemplo Prático

programa
{
    funcao inicio()
    {

        escreva("Bem-vindo ao Portugol Academy")

        escreva("Hoje vamos aprender lógica de programação")

    }
}
