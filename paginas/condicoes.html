Estruturas Condicionais

As estruturas condicionais permitem que o programa tome decisões de acordo com determinadas condições.

O que é uma Condição?

Uma condição é uma comparação que pode resultar em dois valores: verdadeiro ou falso.

Exemplo do dia a dia

SE estiver chovendo → levar guarda-chuva

SENÃO → sair normalmente

Estrutura SE

A estrutura se executa um bloco de código somente se a condição for verdadeira.

Exemplo

inteiro idade

idade = 20

se (idade >= 18)
{
    escreva("Maior de idade")
}

Estrutura SE / SENÃO

Podemos usar senao quando queremos executar um bloco diferente caso a condição seja falsa.

Exemplo

inteiro idade

idade = 16

se (idade >= 18)
{
    escreva("Maior de idade")
}
senao
{
    escreva("Menor de idade")
}

    }
