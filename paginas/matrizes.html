Matrizes

Matrizes são estruturas de dados que armazenam valores em duas dimensões: linhas e colunas.

O que é uma Matriz?

Uma matriz é como uma tabela composta por linhas e colunas. Cada posição da matriz possui dois índices.

Exemplo de matriz 3x3

[0][0] [0][1] [0][2]
[1][0] [1][1] [1][2]
[2][0] [2][1] [2][2]

Declarando uma Matriz

Para declarar uma matriz no Portugol usamos dois conjuntos de colchetes.

Exemplo

inteiro numeros[3][3]

Isso significa uma matriz com 3 linhas e 3 colunas.

Atribuindo Valores

Cada posição da matriz possui dois índices: linha e coluna.

Exemplo

inteiro numeros[2][2]

numeros[0][0] = 10
numeros[0][1] = 20

numeros[1][0] = 30
numeros[1][1] = 40

Usando Matrizes com Repetição

Normalmente utilizamos dois loops para percorrer uma matriz: um para linhas e outro para colunas.

Exemplo

programa
{
    funcao inicio()
    {

        inteiro matriz[3][3]
        inteiro i, j

        para(i = 0; i < 3; i++)
        {
            para(j = 0; j < 3; j++)
            {
                escreva("Digite um número: ")
                leia(matriz[i][j])
            }
        }

        para(i = 0; i < 3; i++)
        {
            para(j = 0; j < 3; j++)
            {
                escreva(matriz[i][j])
            }
        }

    }
}

Estrutura de uma Matriz

Linhas

Representam a primeira dimensão da matriz.

Colunas

Representam a segunda dimensão da matriz.

Índices

Assim como vetores, matrizes começam no índice 0.
